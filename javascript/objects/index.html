About Objects

Objects are a complex data type that let you keep a collection of properties and values in one place. They help you organize code that belongs together. Like in the quiz app, we can collect all of our questions and answers and organize them in one place in javascript.

Objects hold key/value pairs. The keys have to be unique but the values don't, and, they can be any data type.. numbers, strongs, booleans, null, etc.

When an object has a value that is a function, it's called a methond.

      const mammal = {
        numEyes: 2,
        warmBlooded: true,
        evolve: function() {
          console.log("I'm not mutating, I'm evolving.");
          mammal.numEyes++;
        },
      };

      console.log(`mammal has ${mammal.numEyes} eyes`);
      mammal.evolve();
      console.log(`mammal now has ${mammal.numEyes} eyes`);

Getting values looks like this:

const ledZeppelin = {
  singer: 'Robert Plant',
  'guitar player': 'Jimmy Page',
  bass: 'John Paul Jones',
  drums: 'John Bonham'
}

console.log(ledZeppelin.singer);
//or with bracket notation
console.log(ledZeppelin['guitar player']);

This oject has a variable inside it (person) that has a function that returns a self referencial firstName plus lastName to fullName. 'return person' is then returning the value of the 'fullName' function inside person.

  function personMaker() {
  var person = {
    firstName: 'Paul',
    lastName: 'Jones',
    fullName: function() {
      return this.firstName + ' ' + this.lastName;
    },
  };
  return person;
}
