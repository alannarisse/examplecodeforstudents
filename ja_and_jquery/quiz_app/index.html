Welcome to your quiz app helper

Your project will need:

  • Start screen with a start button
  • User is taken through a series of 10 questions with atleast 4 multiple choice answers Per. Should be asked one question after the other. Only be prompted with one question at a time.
  • User Can’t skip questions
  • Be able to see where they are (7 out of 10) and current score (5, correct, 2 incorrect)
  • After answer sumit, receive feedback (good job! Or here s the correct answer then move on to the next answer
  • Final page shows overall score 45/50 etc. Be prompted to start a new game

View the UX/Responsiveness Checklist

Specs and Requirements:

  • Use html forms
  • Use html, css and jquery
  • A11y -- keyboard usable
  • Responsive design (think about mobile, use media queries)

You’ll need:

  • At least 3 sets of 5 questions typed up, any images you want to use
  • Design using pen and paper, them html wireframes
